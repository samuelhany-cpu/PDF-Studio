# PDF Super Studio Pro AI - Icon Assets
# Base64 encoded PNG icons for different resolutions

## NOTE: For actual deployment, you should generate real PNG files from the SVG
## These are placeholder Base64 strings. Use ImageMagick or similar tools:
## 
## Commands to generate from Logo.svg:
## inkscape Logo.svg --export-filename=icon512.png --export-width=512
## inkscape Logo.svg --export-filename=icon256.png --export-width=256
## inkscape Logo.svg --export-filename=icon128.png --export-width=128
##
## Or use online converters: https://cloudconvert.com/svg-to-png
